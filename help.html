Measure exactly 1 liter using logic and strategy.

🎮 Game Overview

You have three jugs:

  • 12L Jug - starts full
  • 7L Jug - empty
  • 5L Jug - empty

Your goal is to get exactly 1 liter using the fewest moves possible.

✨ Features

  • Smooth liquid animations
  • Smart buttons (only valid moves shown)
  • Directional arrows ➡️ ⬅️
  • Victory detection 🎉
  • Responsive design (mobile + desktop)
  • Real-time volume feedback

🎯 How to Play

  1. Click arrows to pour milk between jugs
  2. Only valid moves are allowed
  3. Watch liquid animate as you pour
  4. Reach exactly 1L to win

🕹️ Controls

Button Action
➡️ / ⬅️ 7L Pour to/from 7L jug
➡️ / ⬅️ 5L Pour to/from 5L jug
🍼 FILL 12L Reset game

🧩 Puzzle Logic

This classic puzzle allows only pouring operations:

  • Empty source jug
  • Fill destination jug
  • Or both at once

🏆 Winning Condition

  • Get exactly 1L in any jug
  • Triggers victory animation 🎉
  • Try to beat the optimal move count!
